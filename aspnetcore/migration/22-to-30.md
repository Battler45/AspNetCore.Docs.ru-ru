## <a name="update-the-project-file"></a><span data-ttu-id="eda08-106">Обновляет файл проекта</span><span class="sxs-lookup"><span data-stu-id="eda08-106">Update the project file</span></span>

  ```

  </ItemGroup>
  ```

## <a name="jsonnet-support"></a><span data-ttu-id="eda08-122">Поддержка Json.NET</span><span class="sxs-lookup"><span data-stu-id="eda08-122">Json.NET support</span></span>
